# 🔐 Projeto Tela de Login

Projeto desenvolvido durante o curso de **HTML5 e CSS3** do Curso em Vídeo.

O objetivo foi criar uma interface moderna de **tela de login** utilizando HTML e CSS.

## 📚 Conteúdo aplicado

* Criação de formulários HTML
* Estilização de inputs
* Layout responsivo
* Uso de flexbox
* Centralização de elementos

## 🎯 Objetivo

Praticar a criação de **interfaces de autenticação**, muito comuns em sistemas web.

## 💻 Tecnologias utilizadas

* HTML5
* CSS3

## 📖 Aprendizados

Este projeto permitiu aprender:

* Estrutura de formulários
* Estilização de campos de entrada
* Organização visual de interfaces
