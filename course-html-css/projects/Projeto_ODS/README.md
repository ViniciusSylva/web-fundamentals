# 🌍 Projeto ODS — Redução de CO₂

Este projeto foi desenvolvido como **trabalho acadêmico para a faculdade**, com foco na conscientização sobre a **redução da emissão de dióxido de carbono (CO₂)** e sua relação com os **Objetivos de Desenvolvimento Sustentável (ODS)**.

O projeto apresenta informações sobre sustentabilidade e práticas que podem contribuir para a diminuição do impacto ambiental causado pela emissão de gases do efeito estufa.

## 🌱 Sobre as ODS

As **ODS (Objetivos de Desenvolvimento Sustentável)** fazem parte da Agenda 2030 da ONU e representam um conjunto de metas globais voltadas para o desenvolvimento sustentável, incluindo temas como:

* combate às mudanças climáticas
* preservação ambiental
* consumo responsável
* desenvolvimento sustentável

## 🎯 Objetivo do projeto

O objetivo deste trabalho foi:

* apresentar informações sobre **emissão de CO₂**
* explicar os impactos ambientais causados pelo excesso desse gás
* demonstrar **ações e práticas que ajudam a reduzir a emissão de carbono**

## 📚 Conteúdo aplicado no projeto

Durante o desenvolvimento foram aplicados conceitos de:

* Estruturação de páginas com **HTML5**
* Estilização com **CSS3**
* Organização de conteúdo informativo
* Layout voltado para leitura e apresentação de informações
