# 📱 Projeto Redes Sociais

Projeto desenvolvido durante o curso de **HTML5 e CSS3** do Curso em Vídeo.

Este projeto simula uma interface de celular com links para redes sociais.

## 📚 Conteúdo aplicado

* Posicionamento com CSS
* Uso de **iframe**
* Organização visual de layout
* Simulação de interface mobile

## 🎯 Objetivo

Treinar a criação de interfaces visuais e interativas utilizando HTML e CSS.

## 💻 Tecnologias

* HTML5
* CSS3

## 📖 Aprendizados

* Estrutura de navegação
* Uso de iframes
* Posicionamento de elementos
* Interface simulando dispositivos móveis
